Talk Notes: The Two Solitudes for Hackers

by Tavish Armstrong / tavisharmstrong at gmail dot com / @tavarm

These are notes on a talk I gave at PyCon 2014. I hope this is a useful reference for anyone who saw the talk and wants to learn more about what I spoke about.

If you are interested in what I spoke about in this talk, please, please, get in touch via email. A lot of people mentioned afterwards that they'd be interested in a mailing list or something similar. My hope is that we can gain momentum by working on this together and motivating each other.
