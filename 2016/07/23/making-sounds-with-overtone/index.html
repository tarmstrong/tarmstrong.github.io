Making sounds with overtone

Sat 23 Jul 2016 23:12:00

I’ve been messing about with Overtone, a Clojure interface to SuperCollider to make interesting sounds as a way to combine my interests in programming and music. Overtone has been particularly enjoyable to use because it is (unlike SuperCollider) a programming language that has uses beyond sound synthesis, it is surprisingly easy to set up, and is not under active development (which, for the time being, means that it is stable. I’m looking on the bright side here.)
